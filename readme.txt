=== Description of Cashlesso Payment Gateway ===

This is Cashlesso Payment Gateway for Wordpress . Its Allow you to use Cashlesso payment gateway with the WooCommerce plugin. It is secure to pay online via cashlesso and it is strongly recommended for Indian Payments.

 
=== Installation of Cashlesso Woocommerce Kit ===
1. Ensure that you have latest version of WooCommerce plugin installed.
2. Unzip & Upload contents of the plugin to your /wp-content/plugins/directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.

=== Configuration ===

1. first add the plugin to the worpdress
2. Then Active the Plugin
3. After activation Visit the WooCommerce settings page and click on the Payment Gateways tab of setting page.
4. Click on Cashlesso to Manage the settings. If you do not see Cashlesso in the list at the top of the screen make sure you have activated the plugin in the WordPress Plugin Manager.
5. Fill the Details in the cashlesso fields like: Merchantid, Salt etc . Then click Save button.
